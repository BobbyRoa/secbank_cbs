# Roadmap

## Phase 1: Foundations (0-3 months)

- Define domain model and product catalog scope
- Establish BSP compliance checklist baseline
- Build customer onboarding and KYC workflows
- Implement core ledger and account framework

## Phase 2: Products & Payments (3-6 months)

- Deliver deposit products (CASA, time deposits)
- Implement loan servicing and repayment scheduling
- Integrate internal transfers and payment rails
- Build GL mappings and basic financial reporting

## Phase 3: Risk & Compliance (6-9 months)

- Implement AML transaction monitoring
- Add limits, exposure monitoring, and alerts
- Expand regulatory reporting automation
- Build reconciliation dashboards

## Phase 4: Scale & Optimization (9-12 months)

- High availability and disaster recovery readiness
- Performance tuning and observability automation
- Expand channel integrations and API ecosystem
